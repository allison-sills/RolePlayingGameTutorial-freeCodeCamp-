# JavaScript Role Playing Game Tutorial freeCodeCamp

Welcome to the JavaScript Role Playing Game tutorial! This project was created using FreeCodeCamp's free JavaScript tutorial. The goal of this project is to practice fundamental HTML, CSS, and JavaScript skills.
Project Overview
The completed project is a simple text-based Role Playing Game (RPG) where players can:
* Purchase health and weapons
* Explore a cave
* Fight enemies
  
The game also includes an easter egg and a boss fight for added excitement.
Features

* Text-based Interface: Engage with the game through a simple and intuitive text-based interface.
* Health and Weapons Shop: Purchase health and weapons to prepare for battles.
* Cave Exploration: Navigate through a cave, to encounter enemies along the way.
* Boss Battle: Face off against a challenging boss.
* Easter Egg: Discover a hidden surprise within the game.
Code Explanations
Throughout the code, I have provided detailed annotations to help you understand how the game is built. These comments will guide you through the logic and structure of the code, making it easier to follow and learn from.
